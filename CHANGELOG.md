# 变更日志

所有重要的变更都会记录在这个文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2026-02-06

### 新增
- 🎉 初始版本发布
- 实现随机图片获取功能
- 添加智能图片压缩算法
- 支持多种图片格式（JPEG, PNG, WEBP, BMP, TIFF）
- 内置LRU缓存机制
- 完整的RESTful API接口
- Docker化部署支持
- 健康检查和统计监控
- 灵活的环境变量配置

### 改进
- 使用uv作为包管理器
- 规范化的项目结构
- 完善的异常处理机制
- 详细的日志记录
- 性能优化和资源管理

### 修复
- 图片处理模块导入问题
- JSON序列化兼容性问题
- Docker部署配置问题
